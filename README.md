# YStatusBarHUD
状态栏指示器小框架

## 显示加载成功
```objc
   [YStatusBarHUD showSucess:@"加载成功"];
```

## 显示加载失败
```objc
   [YStatusBarHUD showError:@"加载失败"];
```

## 显示加载
```objc
   [YStatusBarHUD showLoading:@"正在加载"];
```

## 显示信息
```objc
   [YStatusBarHUD showMessage:@"加载消息"];
```
## 隐藏
```objc
   [YStatusBarHUD hide];
```
